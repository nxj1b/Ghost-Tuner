🎛️ Ghost Tuner: Neural Overdrive

Ghost Tuner is a simple yet powerful Windows performance companion designed to eliminate lag, stop screen freezing, and ensure your processor focuses entirely on making your games run at peak performance.

Instead of dealing with complicated Windows settings menus or risky modifications, Ghost Tuner allows you to optimize your operating system with simple clicks. It also includes an advanced background AI module that acts like a private mechanic, listening to system warnings and fixing errors in real time before they can cause game crashes. 🚀

🕹️ Control Panel Guide (What the Buttons Do)
Here is a plain-English breakdown of every single button on the Ghost Tuner interface, written exactly as they appear on your screen:

⚙️ Left Sidebar Actions

🟢 INITIALIZE QUANTUM CORE:
What it does: This is the master start button. Clicking this puts Ghost Tuner into active monitoring mode. It starts tracking your open applications, analyzing your processor speeds, and waking up the AI diagnostics pipeline.
When to use it: Click this as soon as you open the app to begin optimization.

🔴 GHOST GAME BOOST (MANUAL):
What it does: Instantly forces your PC into maximum overdrive. It sets Windows power scaling to high-performance limits, tells your graphics drivers to prioritize direct game processing, disables background data collection, and updates your network settings to lower online match ping.
When to use it: Use this if your game isn't automatically detected by the system and you want to force maximum performance options immediately.

⚪ CREATE SYSTEM RESTORE:
What it does: Snaps a safety backup profile of your current Windows system settings. If any setting adjustment causes unexpected behavior on your specific machine, this ensures you have a clean undo point.
When to use it: Always click this once before applying deep optimization shifts for peace of mind.

⚪ SYSTEM REVIVE:
What it does: Instantly opens the official built-in Windows System Restore wizard tool, allowing you to easily roll back your PC settings to a previous state if needed.
When to use it: Only if you ever need to undo recent operating system setting changes.

⚪ TERMINATE ALL ENGINES:
What it does: Instantly stops all background monitoring scripts, turns off the optimization rules, and safely returns your Windows system profiles back to their standard stock defaults.
When to use it: When you are done gaming and want to switch your computer back into normal, everyday power-saving mode.

🔵 NEURAL PURGE SWEEP:
What it does: Performs a deep digital clean-up across your machine. It flushes your internet address lookup data to clean up network routing, forces background programs to empty their hoarded memory allocations, and empties bloated hidden temporary storage cache locations alongside your Recycle Bin.
When to use it: Click this whenever your PC starts feeling generally sluggish or slow during standard desk work.

🧽 WIPE SHADER CACHE:
What it does: Deletes old, stale temporary graphical files saved by your graphics card. Over time, these files can become corrupt and cause your games to experience sudden 1-second stutters or hitching when you turn camera angles.
When to use it: Use this if your game matches feel laggy or choppy despite having high average frames-per-second (FPS).

💾 FIX PAGEFILE SIZE:
What it does: Locks your computer's virtual hard drive memory to a stable, fixed size. Normally, Windows constantly shrinks and expands this file while you play, causing sudden performance drops mid-game; locking it removes those structural speed bumps.
When to use it: Apply this once to enjoy a smoother, more consistent frame layout during long gaming sessions.

🎚️ Custom Sliders & Toggles

⚡ Dynamic Core Cap (Slider):
What it does: Lets you control the maximum allowed speed capacity of your CPU. When gaming mode is inactive, lowering this cap allows your PC to run much cooler and save electricity. The moment a game is detected, Ghost Tuner automatically bypasses this limit and unlocks 100% full speed.
☀️ / 🌙 (Theme Toggle):
What it does: Instantly switches the entire interface layout look between a clean, bright Light Mode and a comfortable, sleek Dark Mode.
🧠 AI AUTO-OPTIMIZER (Switch):
What it does: Located at the top right of the application window. When switched on, it allows the Gemini AI engine to actively analyze live computer error log files, automatically deploying secure hotfixes behind the scenes to keep games from crashing. Turn it off if you want to use the performance buttons purely manually without automated intervention.
